# ClipFlow — AI Agent Context

## What is this?

macOS native clipboard manager. Swift/SwiftUI, macOS 14+, Swift Package Manager.
Menu bar app with floating panel (⌥V toggle), keyboard-first UX (⌥1-9 paste, ESC dismiss).

## Architecture

```
ClipFlowApp → AppState (orchestrator, @MainActor)
               ├── ClipboardMonitor (0.5s polling, background image processing)
               ├── HotkeyManager (Carbon HIToolbox, dynamic register/unregister)
               ├── FloatingPanelController (NSPanel, orderOut/orderFront lifecycle)
               ├── PanelView (SwiftUI, 320x420 floating panel)
               └── MenuBarView (SwiftUI, MenuBarExtra)

Data model: ClipItem → ClipContent enum (text | image | richText | file)
Paste: PasteHelper (CGEvent ⌘V simulation with 50ms timing delays)
Image: ImageProcessor (PNG storage, 2048px cap, 96px thumbnail, aspect-fill crop)
```

## Key Invariants

- **isPasting guard**: AppState.addItem() skips items while paste is in flight (prevents self-capture loop). Uses cancellable Task with 500ms window.
- **Transient filtering**: ClipboardMonitor skips org.nspasteboard.TransientType/ConcealedType/AutoGeneratedType (password managers).
- **Image pipeline**: Pasteboard TIFF/PNG/JPEG → NSImage → downscale to 2048px → PNG Data storage → 96px thumbnail. All on background DispatchQueue.
- **Panel lifecycle**: orderOut (not close) to keep NSHostingView alive. Avoids SwiftUI rebuild on every toggle.
- **Hotkey lifecycle**: ⌥V always registered. ⌥1-9 + ESC only registered when panel is visible.

## ClipContent Enum

```swift
enum ClipContent {
    case text(String)                              // fully implemented
    case image(Data, thumbnail: NSImage, size: CGSize)  // fully implemented
    case richText(NSAttributedString)              // placeholder — assertionFailure in debug
    case file(URL, name: String)                   // placeholder — no-op in paste
}
```

## Build & Run

```bash
swift build && .build/debug/ClipFlow &
```

Requires macOS Accessibility permission for CGEvent paste simulation.
